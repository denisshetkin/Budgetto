# UX Structure

## Main screens
1. Home (donut chart + balance + quick +/-)
2. Add Transaction (income/expense, amount, category, date, note)
3. Transactions list (filters by time/category)
4. Categories (system + custom management)
5. Budgets (weekly/monthly, by category)
6. Reports (time + categories)
7. Settings (currency, sync, account)

## Primary flow
- Home -> tap + or - -> enter amount -> choose category -> save

## Key components
- Donut chart with category icons around
- Period toggle (week/month)
- Large amount display
- Soft cards (no hard borders)
- Thin outline category icons
