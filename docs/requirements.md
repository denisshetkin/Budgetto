# Requirements

## Scope (MVP)
- One global balance (no separate wallets/accounts)
- Income and expense tracking
- Categories: system + user can add/edit/delete any
- Periods: weekly and monthly
- Reports: by time and by categories
- Optional sync: Apple/Google sign-in only

## Behaviors
- Default date is "today" for new transaction
- Date can be edited for past entries
- Categories are fully user-managed
- Sync is disabled by default; user can enable in settings

## Non-goals (MVP)
- No reminders
- No multi-currency wallets (single selected currency)
- No recurring transactions
